# Spotify API

API in development for data requests in the Official API of spotify.

## SETUP

```bash
  # install dependencies
  npm install

  # start build
  npm run build

  # start application
  npm start

  # start eslint
  npm run lint

  # start eslint fix
  npm run lint:fix

  # start test with jest
  npm run test

  # listen test with jest
  npm run test:watch

  # listen test on changed files
  npm run test:staged

  # starts test coverage
  npm run test:coverage

  # setup husky
  npm run prepare
